# Import Conventions

There are 4 categories of modules which are Node.js modules, dolphinOS modules, 3rd party modules, and own modules.

Import order of categories and modules should be as followed.
An empty line should be inserted between the categories.
In selective imports, if more than 1 module is imported, modules should be on separate lines.

## Node.js Modules

const path = require("path");
const fs = require("fs");
const http = require("http");
const https = require("https");
