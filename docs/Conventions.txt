# Import Conventions

There are 4 categories of modules which are Node.js modules, dolphinOS modules, 3rd party modules, and own modules.

While requiring modules, an empty line should be inserted between the modules.
In addition, under each category the order below should be followed.

## Node.js Modules

const path = require("path");
const fs = require("fs");
const http = require("http");
const https = require("https");
